<p>jifbar is a Simple Status Bar Used with DWM on Arch Linux Systems</p>

<h1>Dependencies</h1>
<ul>

<li>pacman-contrib package</li>

</ul>

<h1>Install</h1>
<ol>

<li>run make and then make install as root</li>

</ol>

<h1>Uninstall</h1>
<ol>

<li>run make uninstall as root</li>

</ol>
